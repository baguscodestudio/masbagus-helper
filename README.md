# masbagus-helper

Bot for my community discord

## Available commands

- `/prunespecial [amount]`
- `/prune [amount]`
- `/iseng`
- `/ping`

## Installation

1. Make sure you have node js v16+ installed. Run `npm install`
2. Create a .env folder and put this inside

```env
TOKEN =
GUILD_ID =
OWNER_DISCORD =
```

3. Then run with `node main.js`
